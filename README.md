# Linux System Monitoring

This repository contains a set of practical projects focused on Linux system monitoring, automation, and analysis.

The work is divided into two main parts:

- Bash scripting and system research
- Real-time monitoring and observability

## Overview

The project demonstrates how to:

- automate system tasks using Bash
- generate and analyze system data
- work with filesystem structures
- process logs
- monitor system performance in real time

## Part 1 — Bash and System Analysis

Covers fundamental scripting and system inspection:

- system information collection
- colored output formatting
- filesystem analysis
- file and directory generation
- cleanup automation
- log generation (nginx format)
- log parsing using awk

## Part 2 — Monitoring and Observability

Focuses on real-time system monitoring tools:

- log analysis with GoAccess
- metrics collection with Prometheus
- data visualization with Grafana
- system load testing (CPU, RAM, disk)
- network performance testing (iperf3)
- building custom exporters

## Tools Used

- Bash
- awk
- nginx
- GoAccess
- Prometheus
- Grafana
- iperf3
- stress
- cron

## Environment

- Ubuntu Server (virtual machines)
- VirtualBox or similar virtualization tool

## Skills Demonstrated

- Linux system administration
